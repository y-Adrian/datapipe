# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-08

### Added
- Initial release
- Core data pipeline functionality
- Buffer management with auto-expansion
- Base64 encoding/decoding
- Hex encoding/decoding
- XOR encoding/decoding
- Platform abstraction layer
- Multi-platform support (x86_64, ARM64, RISC-V64)
- CMake build system
- Cross-compilation toolchain files
- GitHub Actions CI/CD pipeline
- Unit tests for all modules
- Docker test environments

### Fixed
- Header include path configuration
- Missing `DATAPIPE_EXPORT` annotations for symbol visibility
- CI configuration package installation commands
